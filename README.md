# Google Search Operators Master Directory (48 Examples)

Interactive single-page web application listing 48 Google Search operators, power combos, and OSINT dorks with real-time search, category filtering, and direct test links.

Інтерактивний веб-додаток (SPA), що містить 48 операторів пошуку Google, комбінацій та OSINT-дорків із функцією швидкого пошуку, фільтрацією за категоріями та кнопками копіювання.

## 🚀 Live Demo

[Open Interactive Directory](https://nopainnoreason.github.io/-Search-Operators-Master-Directory/index.html)

## 🛠 Features / Можливості

- **48 Operators & Combos**: Complete coverage from basic syntax to advanced search dorks.
- **Bilingual Interface**: Descriptions and examples in English and Ukrainian (`EN` / `UA`).
- **Instant Filtering**: Live search by name, description, or keyword + category selector.
- **Dark / Light Theme**: Automatic preference detection with a manual toggle switch.
- **One-click Actions**: Direct `Copy` button for examples and `Google It` test links.
- **Zero Dependencies**: Pure HTML5, Tailwind CSS (via CDN), and Vanilla JavaScript.

## 📊 Included Categories / Категорії

| Category | Description (EN) | Опис (UA) |
| :--- | :--- | :--- |
| `basic` | Core boolean and exact match operators | Базові логічні оператори та точний збіг |
| `technical` | Domain, filetype, and index targeting | Пошук за доменами, типами файлів та індексом |
| `content` | Title, URL, anchor, and text matching | Пошук у заголовках, посиланнях та тексті |
| `advanced` | Proximity, numerical ranges, widgets | Відстань між словами, діапазони та віджети |
| `deprecated` | Historical and disabled operators | Застарілі та відключені оператори |
| `combo` | Security dorks and advanced audit queries | Комплексні OSINT-дорки та аудит безпеки |
